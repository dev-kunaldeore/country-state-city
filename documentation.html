3. /get-cities-by-country

Endpoint

GET /get-cities-by-country?country_code={COUNTRY_CODE}

Request

country_code

Response

  • Status Code: 200 OK
  • Content Type: application/json

[
  {
    "id": 57584,
    "name": "Abhaneri",
    "iso2": "IN"
  },
  {
    "id": 57585,

Error Handling

If an error occurs during the processing of a request, the API will respond with an error message and an appropriate status code.

  • Status Code: 400 Bad Request
  • Content Type: text/plain

Example Error Response

Missing country_code parameter
  • Status Code: 500 Internal Server Error
  • Content Type: application/json

Example Error Response Body

{
  "error": "Error message details..."
}

Notes

  • All responses are in JSON format.
  • Ensure that the required data files (country.json, states.json, cities.json) are available at the specified file paths (../data/).
